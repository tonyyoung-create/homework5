## ✨ 核心改進

### 問題 1: 評分全為零
```python
# ❌ 修復前 - 依賴特徵
if perplexity > 0:
    score += ...  # 特徵失敗時不執行

# ✅ 修復後 - 純統計
vocab_score = calculate_ttr()  # 總是有值
score += vocab_score
```

### 問題 2: 依賴缺失
```python
# ❌ 修復前 - 需要 NLTK 標籤器
try:
    tags = nltk.pos_tag(words)  # 經常失敗
except:
    return error

# ✅ 修復後 - 不使用 NLTK
function_words = {'the', 'and', ...}
ratio = count(w for w in words if w in function_words)
```

### 問題 3: 特徵計算超時
```python
# ❌ 修復前 - 複雜計算
perplexity = compute_with_model()  # 對長文本失敗

# ✅ 修復後 - 簡單統計
consistency = 1 - (std_dev / mean_len)  # 快速可靠
```
