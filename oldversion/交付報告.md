## 🔍 核心改進詳解

### 問題 1: 評分全為 0
**修復前**:
```python
# ❌ 如果任何特徵失敗，評分為 0
ai_score = 0
if perplexity > 0:  # 失敗時這不執行
    ai_score += ...
# ai_score = 0 → 100% 人類
```

**修復後**:
```python
# ✅ 純統計，無依賴，100% 可靠
ai_score = 0
vocab_score = max(0, min((vocab_ratio - 0.5) / 0.3, 1)) * 0.35
ai_score += vocab_score  # 總是有值
```

### 問題 2: NLTK 資源缺失
**修復前**: 依賴 `averaged_perceptron_tagger_eng` (經常不存在)  
**修復後**: 完全拋棄 NLTK，使用簡單的英文詞表

### 問題 3: 特徵計算超時
**修復前**: Perplexity 計算對長文本索引失敗  
**修復後**: 不計算困惑度，使用流暢性指標替代
